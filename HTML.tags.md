# Modern HTML Tags – Hebrew Descriptions

> מבוסס על MDN Web Docs – גרסה עדכנית של תקן HTML Living Standard.
> כולל תגיות פעילות בלבד (ללא מיושנות).

---

## Document structure

- **`<!DOCTYPE>`** — מציין את סוג המסמך וגרסת HTML.
- **`<html>`** — תגית השורש של כל מסמך HTML.
- **`<head>`** — מכילה מידע מטא, קישורים לקבצים חיצוניים, סקריפטים וכו׳.
- **`<title>`** — מגדירה את כותרת הדף המוצגת בטאב הדפדפן.
- **`<base>`** — מגדירה כתובת URL בסיסית לכל קישורי הדף.
- **`<link>`** — משמשת לקישור לקובצי CSS או משאבים חיצוניים.
- **`<meta>`** — מספקת מידע על הדף (תיאור, קידוד, שפה וכו׳).
- **`<style>`** — מכילה קוד CSS בתוך הדף.
- **`<body>`** — מייצגת את כל תוכן הדף הנראה למשתמש.

---

## Text content

- **`<h1>`–`<h6>`** — כותרות לפי דרגה (מחשיבות עליונה לנמוכה).
- **`<p>`** — פסקת טקסט רגילה.
- **`<hr>`** — קו אופקי להפרדת תוכן.
- **`<pre>`** — טקסט עם שמירת רווחים ושורות כפי שנכתב.
- **`<blockquote>`** — ציטוט ארוך מטקסט אחר.
- **`<ol>` / `<ul>` / `<li>`** — רשימות ממוספרות, לא-ממוספרות ופריטי רשימה.
- **`<dl>` / `<dt>` / `<dd>`** — רשימת הגדרות (מונח והסבר).
- **`<figure>` / `<figcaption>`** — איור/תמונה עם כיתוב נלווה.
- **`<div>`** — בלוק כללי ללא משמעות סמנטית.
- **`<main>`** — החלק המרכזי של הדף (פעם אחת בלבד).
- **`<section>`** — אזור עצמאי בתוכן, כמו פרק או נושא.
- **`<article>`** — יחידת תוכן עצמאית (מאמר, פוסט, תגובה).
- **`<aside>`** — תוכן משני או צדדי, כמו סרגל צד.
- **`<header>`** — כותרת עליונה של מסמך או קטע.
- **`<footer>`** — כותרת תחתונה עם פרטים מסכמים או קישורים.
- **`<nav>`** — אזור ניווט ראשי של האתר.

---

## Inline text semantics

- **`<a>`** — קישור לדף אחר או לעוגן פנימי.
- **`<em>`** — טקסט מודגש במשמעות הדגשה סמנטית.
- **`<strong>`** — טקסט בעל חשיבות גבוהה (מופיע לרוב מודגש).
- **`<small>`** — טקסט משני או הערה קטנה.
- **`<cite>`** — ציון מקור יצירה.
- **`<q>`** — ציטוט קצר בתוך שורה.
- **`<dfn>`** — מגדיר מונח חדש.
- **`<abbr>`** — קיצור או ראשי תיבות עם משמעות מוסברת.
- **`<time>`** — מציין זמן או תאריך בפורמט נגיש.
- **`<code>`** — קטע קוד או ביטוי טכני.
- **`<var>`** — מייצג משתנה בקוד או נוסחה.
- **`<samp>`** — פלט מדומה של תוכנית.
- **`<kbd>`** — קלט מקלדת או קיצור דרך.
- **`<mark>`** — טקסט מודגש לציון חיפוש או רלוונטיות.
- **`<span>`** — אלמנט inline כללי ללא משמעות סמנטית.
- **`<br>`** — ירידת שורה.
- **`<wbr>`** — נקודת שבירת שורה אפשרית.
- **`<b>`** — טקסט מודגש ויזואלית (לא סמנטית).
- **`<i>`** — טקסט נטוי (למשל מונח זר).
- **`<u>`** — קו תחתי, לרוב להדגשה גרפית בלבד.
- **`<sub>` / `<sup>`** — טקסט תת-כתב או על-כתב.
- **`<bdi>` / `<bdo>`** — שליטה בכיווניות טקסט RTL/LTR.

## Media and embedding

- **`<img>`** — מציג תמונה.
- **`<picture>` / `<source>`** — תמונות רספונסיביות לפי רוחב מסך או פורמט.
- **`<video>` / `<audio>`** — הצגת וידאו או שמע.
- **`<track>`** — כתוביות או תרגום למדיה.
- **`<iframe>`** — הטמעת דף אינטרנט בתוך דף אחר.
- **`<embed>`** — משאב מוטמע חיצוני (כמו PDF או מדיה אחרת).
- **`<object>` / `<param>`** — הטמעה גנרית של מדיה או יישום.
- **`<canvas>`** — אזור ציור גרפי הנשלט ע״י JavaScript.
- **`<svg>`** — גרפיקה וקטורית ניתנת לעיצוב ו-JS.

---

## Forms

- **`<form>`** — טופס לאיסוף נתונים מהמשתמש.
- **`<input>`** — שדה קלט במגוון סוגים.
- **`<label>`** — תווית תיאורית לשדה קלט.
- **`<button>`** — כפתור לשליחה או פעולה.
- **`<select>` / `<option>` / `<optgroup>`** — תיבת בחירה נפתחת.
- **`<textarea>`** — אזור טקסט רב-שורות.
- **`<fieldset>` / `<legend>`** — קבוצת שדות עם כותרת.
- **`<datalist>`** — רשימת הצעות קלט לשדה.
- **`<output>`** — מציג תוצאה מחושבת מטופס.
- **`<progress>`** — מציין התקדמות תהליך.
- **`<meter>`** — מד ערך בטווח מסוים.

---

## Interactive elements

- **`<details>` / `<summary>`** — תוכן מתקפל/נפתח.
- **`<dialog>`** — חלון דיאלוג אינטראקטיבי.
- **`<menu>`** — תפריט פקודות או אפשרויות.
- **`<slot>` / `<template>`** — תבניות לתוכן דינמי (Web Components).
- **`<script>`** — הטמעת או טעינת קוד JavaScript.
- **`<noscript>`** — תוכן שיוצג אם JavaScript מושבת.

---

## Table elements

- **`<table>`** — טבלה להצגת נתונים.
- **`<caption>`** — כותרת לטבלה.
- **`<thead>` / `<tbody>` / `<tfoot>`** — חלוקת חלקי הטבלה.
- **`<tr>`** — שורת טבלה.
- **`<th>` / `<td>`** — תא כותרת ותא רגיל.
- **`<col>` / `<colgroup>`** — הגדרת עמודות ויישורן.

---

## Scripting and data

- **`<data>`** — מצמיד ערך נתון לטקסט מוצג.
- **`<template>`** — תוכן נסתר המשמש לשכפול דינמי.
- **`<slot>`** — מקום לתוכן שיוזרק מבחוץ בקומפוננטות מותאמות.
